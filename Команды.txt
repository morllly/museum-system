ИНСТРУКЦИЯ
1) В корне проекта выполнить
	##добавить папку laradock##
	git submodule add https://github.com/LaraDock/laradock.git laradock

2) зайти в папку laradock
назовите в .env
COMPOSE_PROJECT_NAME=museum-system

убедитесь docker-compose.yml:
PHP_VERSION=8.0

3) запустить контейнеры
	docker-compose up -d nginx mysql phpmyadmin
4) войти в контейнер рабочей области:
	docker exec -it laradock-workspace-1 bash
	docker-compose exec workspace bash
5) выпонить
	##добавить папку vendor##
	composer install (в случае если не сработает, то composer install --no-plugins)
    ##добавить папку node_modules##
    npm install


------------------------------------------

Использование phpMyAdmin#
Запустите phpMyAdmin Container (phpmyadmin) с docker-compose up с помощью команды.
docker-compose up -d mysql phpmyadmin

Откройте браузер и посетите localhost на порту 8081: http://localhost:8081, use server: “mysql”, user: “root” и password: “root для настройки mysql по умолчанию.
